Community

There are several places where one can talk about Ironclad, be it for development, talking about bugs, or just chitty chatter. This page lists official channels of communication, that is, channels that comply with the conduct and communication guidelines of the project.

Matrix

There's an official matrix server hosted at ironclad:matrix.org.

Discord

Discord and privacy

Discord is the only platform in this list that is not free software. Along with that, Discord has a long history of privacy issues. Please consider joining the Matrix server instead.

The official discord server can be found here.

Mailing lists

Ironclad has several mailing lists hosted on GNU savannah:

  • bug-ironclad@nongnu.org for bug reporting.
  • ironclad@nongnu.org for general project discussion.

Communication guidelines

Ironclad's uses the GNU Kind Communications Guidelines.

Complicance to these guidelines should be enforced at the discretion of the community's moderation, with enforcement implemented as seen fit. Endorsement of communities by Ironclad will be done following these guidelines.
