1 Installation and target support ¶

This chapter delves on what hardware configurations Ironclad supports and how to get Ironclad running on those platforms.

Ironclad hardware support is divided in targets, which are combos of an architecture and board that are supported in tandem, along with any accompaning hardware, some hardware is supported regardless of the underlying target, Common devices.
