8.3 Kernel Address SANitizer (KASAN) ¶

Ironclad supports KASAN, which is useful for debugging memory use by the kernel. KASAN can be configured to be used by being built with --enable-kasan. KASAN comes with a big performance penalty, so only use it for debugging.

KASAN errors will be printed to the architectural debug output and will not result in kernel panics.
