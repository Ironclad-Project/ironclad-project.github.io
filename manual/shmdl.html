9.84 shmdl ¶

int shmdt(const void *shmaddr);

This syscall detaches the shared memory segment located at shmaddr from the calling process.

The syscall returns 0 on success or -1 on failure, with the following errno:

EINVAL

The passed shmaddr is not valid.
