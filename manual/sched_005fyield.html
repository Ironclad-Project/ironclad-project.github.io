9.21 sched_yield ¶

int sched_yield(void);

This syscall relinquishes execution of the caller thread. Its up for the kernel how far in the queue of execution this thread will go.

This syscall returns 0 always, as it never fails, this is done for compatibility with POSIX instead of having a void return type.
